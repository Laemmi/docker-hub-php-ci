# docker-hub-php-ci
Image for php ci and php development.

## Includes
- Linux
    - git
    - less
    - libfreetype6-dev
    - libicu-dev
    - libjpeg62-turbo-dev
    - libldap2-dev
    - libmagickwand-dev
    - libpng-dev
    - libtidy-dev
    - locales 
    - openssh-client
    - rsync
    - ssh
    - unzip

- php 7.2
    - bcmath
    - exif
    - gd
    - intl
    - ldap
    - mysqli
    - opcache
    - soap
    - sockets
    - tidy
    - zip
    - imagick
    - xdebug
- Apache Webserver
    - headers
    - rewrite
    - ssl
    
# Usage
You can use this image for local php development, deployment, unittests, continuous integration.
For local php development have a look to https://github.com/Laemmi/docker-local-php-dev
